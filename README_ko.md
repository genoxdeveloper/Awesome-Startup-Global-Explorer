# Startup-Global: 글로벌 스타트업 생태계 탐색기

![License](https://img.shields.io/github/license/Genox-developer/Startup-Global?color=blue)
![Python](https://img.shields.io/badge/python-3.11+-blue.svg)

**190개국 100개 이상의 산업에서 32,500개가 넘는 스타트업 자금 조달, 지원금, 액셀러레이터 및 클라우드 크레딧을 발견하세요.**

Startup-Global은 전 세계 창업자들이 자본, 인프라 및 성장 기회에 접근할 수 있도록 돕기 위해 **Genox Holdings**가 개발한 오픈 소스 확장형 플랫폼입니다.

---

## 🌟 주요 기능
1. **초대규모 커버리지**: 190개국 32,500개 이상의 기회를 인덱싱합니다.
2. **실시간 크롤러 엔진**: VC, 엔젤 네트워크, 정부 포털 데이터를 수집하는 강력한 백엔드.
3. **고급 필터링**: 국가, 산업, 펀딩 단계, 조직 유형별 정밀 검색.
4. **글로벌 다국어 지원**: 17개 언어로 완벽히 번역됩니다.

자세한 기여 가이드는 [CONTRIBUTING.md](CONTRIBUTING.md)를 참고하세요.
